# MetalColor
Automatic Real Time Colorization with CNN on iOS
## Usage
1. Run it.
2. Put your camera againt gray scaled image or video.
3. Enjoy
## Only tested on iPhone7/iOS 10, I don't know if it supports lower hardware...
